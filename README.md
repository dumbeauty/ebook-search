# 📖 书海 · 电子书聚合搜索

一个简洁的电子书聚合搜索工具，输入书名即可同时检索9个主流书源，省去逐一查找的麻烦。

## ✨ 功能

- 🔍 一次搜索，同时覆盖9个书源
- 🚀 一键打开全部书源
- 📱 响应式设计，手机电脑均可使用
- 🎨 界面简洁，无广告

## 📚 收录书源

| 书源 | 特色 |
|------|------|
| Saltyleo | 综合聚合，主流首选 |
| 安娜的档案 | 英文书及学术资源 |
| Z-Library | 海量书籍 |
| 鸠摩搜书 | 中文书源，格式多样 |
| 书籍知识库 | 中文书籍收录较全 |
| 博看书苑 | 国内杂志期刊 |
| 书格 | 古籍善本 |
| 首都图书馆 | 英文绘本及馆藏 |
| 国家智慧教育平台 | 各科教材 |

## 🚀 部署

本项目为纯静态页面，只有一个 `index.html` 文件。

**Vercel 部署（推荐）：**
1. Fork 或克隆本仓库
2. 前往 [vercel.com](https://vercel.com) 导入仓库
3. 直接 Deploy，无需任何配置

## 📌 声明

本工具仅聚合各平台公开搜索链接，不存储任何书籍文件。
